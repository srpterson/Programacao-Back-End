Foi desenvolvida uma API de livros utilizando Node.js e Express. Cada livro possui os atributos titulo, autor, genero e lido. Foram implementadas as rotas básicas de CRUD: GET /livros, GET /livros/:id, POST /livros, PUT /livros/:id e DELETE /livros/:id. Além disso, foi criada a rota especial GET /livros/nao-lidos, que retorna apenas os livros cujo campo lido é false. A ordem das rotas foi organizada para evitar conflito entre a rota especial e a rota com parâmetro :id, conforme exigido no enunciado.
