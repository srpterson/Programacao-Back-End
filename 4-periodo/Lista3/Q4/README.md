Foi desenvolvida uma API de músicas utilizando Node.js e Express. Cada música possui os atributos titulo, artista, album e duracao. Foram implementadas as rotas de CRUD: GET /musicas, GET /musicas/:id, POST /musicas, PUT /musicas/:id e DELETE /musicas/:id. Além disso, foi criada uma funcionalidade extra de filtro por artista com query string, utilizando a rota GET /musicas?artista=nome. A API utiliza express.json() para leitura do corpo das requisições, req.params para identificação de recursos por ID e req.query para filtragem.
