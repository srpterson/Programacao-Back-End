Foi desenvolvida uma API de medicamentos utilizando Node.js e Express. Cada medicamento possui os atributos nome, dosagem, frequenciaUso e horariosAdministracao. Foram implementadas as rotas básicas de CRUD: GET /medicamentos, GET /medicamentos/:id, POST /medicamentos, PUT /medicamentos/:id e DELETE /medicamentos/:id. A API utiliza express.json() para leitura do corpo das requisições e req.params para identificação de recursos por ID. As informações armazenadas podem ajudar o paciente a organizar melhor sua rotina de medicação ao concentrar dados sobre dosagem, frequência e horários de administração.
