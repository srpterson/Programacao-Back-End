Foi desenvolvida uma API de filmes utilizando Node.js e Express. Cada filme possui os atributos titulo, diretor, ano e nota. Foram implementadas as rotas básicas de CRUD: GET /filmes, GET /filmes/:id, POST /filmes, PUT /filmes/:id e DELETE /filmes/:id. Como funcionalidade extra, foi criada uma rota com query string para filtrar filmes por nota mínima, utilizando GET /filmes?nota=valor. A implementação utiliza express.json() para leitura do corpo das requisições, req.params para parâmetros de rota e req.query para filtros por query string.
