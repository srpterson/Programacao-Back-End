Foi desenvolvida uma API de tarefas utilizando Node.js e Express. Cada tarefa possui descricao, status e dataCriacao. Foram implementadas as rotas de CRUD: GET /tarefas, GET /tarefas/:id, POST /tarefas, PUT /tarefas/:id e DELETE /tarefas/:id. Além disso, foi criada a rota especial PATCH /tarefas/:id/concluir, responsável por atualizar apenas o status da tarefa para "concluida" sem exigir o envio do objeto completo. A API utiliza express.json() para leitura do corpo das requisições e req.params para identificação do recurso.
