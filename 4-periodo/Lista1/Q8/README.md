Foram criados dois projetos equivalentes para demonstrar os dois sistemas de módulos do Node.js. No Projeto A, foi utilizado CommonJS, com arquivos .cjs, require() e module.exports. No Projeto B, foi utilizado ECMAScript Modules, com "type": "module" no package.json e sintaxe import/export. Ambos produziram o mesmo resultado, diferenciando-se apenas pela forma de importação e exportação dos módulos.
