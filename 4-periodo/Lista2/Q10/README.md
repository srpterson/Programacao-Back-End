Foi criada uma função arrow chamada avaliarIMC que recebe múltiplos objetos contendo nome, peso e altura por meio do operador rest. Para cada pessoa, foi utilizado destructuring para acessar os dados, calculado o IMC com a fórmula peso / altura² e definida uma classificação com base no valor obtido. A função retorna um array de strings formatadas com template literals, e no arquivo index.js foram realizados testes exibindo os resultados no console.
