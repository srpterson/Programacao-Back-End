Foi criada uma função arrow chamada criarServidor que recebe um objeto de configuração e utiliza destructuring com valores padrão para host, porta e seguranca. Para permitir chamadas sem argumentos, foi definido um objeto vazio como valor padrão do parâmetro. A função retorna uma string formatada com template literals, e no arquivo index.js foram realizados testes com chamadas completas, parciais e sem argumentos.
