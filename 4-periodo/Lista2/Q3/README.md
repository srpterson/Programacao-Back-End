Foi criado um arquivo alunos.js exportando um array de objetos com as propriedades nome e notas. No arquivo relatorio.js, foi implementada uma função arrow que utiliza map() para gerar um novo array de strings, destructuring para acessar nome e notas, reduce() para somar as notas e calcular a média, e template literals para formatar a saída no padrão solicitado. No arquivo index.js, o relatório foi exibido no console.
