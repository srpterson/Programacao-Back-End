Foi criada uma função arrow chamada gerarRelatorio que recebe múltiplos usuários por meio do operador rest. Cada usuário possui nome, idade e um array de compras. A função utiliza destructuring para acessar as propriedades do usuário e reduce() para somar os valores do array de compras, obtendo o total gasto por cada pessoa. Por fim, retorna um array de strings formatadas com template literals, e no arquivo index.js foram realizados testes exibindo o relatório no console.
