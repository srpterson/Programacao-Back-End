Foi criado um arquivo pedidos.js exportando um array de pedidos contendo id, cliente e total. No arquivo status.js, foi implementada uma função arrow que utiliza map() para gerar um novo array de objetos e o operador spread para copiar os dados originais, adicionando a propriedade status. O status foi definido com operador ternário, sendo "APROVADO" para pedidos com total maior ou igual a 200 e "PENDENTE" nos demais casos. No index.js, os resultados foram exibidos no console.
