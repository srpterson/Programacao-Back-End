Foi criada uma função arrow chamada processarPessoas que recebe múltiplos objetos de pessoa utilizando o operador rest. Cada objeto contém nome, idade e altura. A função utiliza destructuring para acessar as propriedades, operador ternário para identificar se cada pessoa é maior ou menor de idade, e template literals para montar as mensagens finais. No arquivo index.js, a função foi importada e testada com diferentes pessoas, exibindo um array de strings no console.
